# VPNhelper-Windows
本工具能在你第一次使用Windows系统连接到VPN服务器时起到一些帮助。适用于你已经配置好设置，界面也显示已经连接，但仍然无法代理网络时。（VPN类型包括PPTP和L2TP，其他类型暂时不知是否支持）


*警告：本程序需要在管理员权限下运行，否则会出现权限不足的问题。
*运行完成后请重启您的Windows系统，届时重新连接即可。
*若程序无法运行，您也可以前往我们的Wiki界面根据操作说明手动操作。
